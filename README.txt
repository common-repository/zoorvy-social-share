Simple Social Sharing buttons for WordPress that dosent effect the pageload speeds.

== Description ==

Simple Social Sharing buttons for WordPress is the best social sharing plugin with responsive and fast buttons that dosent effect the pageload speeds. You can add Facebook, Twitter, Google+, Linkedin, Reddit, Pocket buttons to your post.

jQuery is required for resposive buttons to work. jQuery plugin is not included in this plugin.


== Installation ==


1. Upload `zoorvy-social-share` folder to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Choose your preference in Settings->Zoorvy Social Share
